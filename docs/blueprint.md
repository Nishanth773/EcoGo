# **App Name**: ZYRH - Green Logistics

## Core Features:

- Shipment Tracking Dashboard: Real-time shipment tracking and overview for customers.
- Eco-Route Optimization: AI powered delivery route planning tool that optimizes routes based on Google Maps API, Weather APIs and shipment data for fuel efficiency and minimal carbon footprint.
- Driver Leaderboard: Gamified leaderboard to rank drivers based on EcoScore ($CO_{2}$ reduction), promoting eco-friendly driving practices.
- Firebase Authentication: Secure user login system via Firebase Auth.
- Real-time Notifications: Push notifications via Firebase Cloud Messaging to alert drivers about new, eco-friendly routes and shipment updates.
- Data Storage and Sync: Firestore to store and synchronize shipment requests, delivery data, and user information.
- EcoMetrics Dashboard: Display data about driver's fuel usage, carbon reduction, and ranking to the user.

## Style Guidelines:

- Primary color: Forest green (#388E3C), reflecting environmental friendliness and sustainability.
- Background color: Light green (#E8F5E9), provides a clean and natural backdrop.
- Accent color: Blue (#1976D2), used for interactive elements to ensure visibility.
- Body and headline font: 'PT Sans', a modern, readable sans-serif for clear UI text.
- Use minimalist line icons with rounded corners to represent shipments, routes, and eco-metrics.
- Clean, card-based layout for dashboard elements to provide a clear overview of key information.
- Subtle transitions and animations for loading new routes and updating eco-scores to provide engaging feedback.
